## 1. 기획 의도 (왜 필요한가?)

- 남들과 경쟁하기를 원하는 플레이어들을 위해 끝없는 도전 스테이지를 제공하고, 누적 처치 수(점수)를 바탕으로 한 랭킹 경쟁을 유도하여 게임의 리텐션을 높이기 위함.

## 2. 선행 조건 및 자원 소모 (언제, 무엇을 써서 하는가?)

- **해금 조건:** 최종 보스 처치 후 '재돌입 결의(트리거 5번)' 선택 시 전용 모드 활성화
    
- **소모 자원:** 별도의 재화 소모 없음 (이번 게임에서 획득한 무기 레벨, 패시브, 현재 체력 및 잔탄 등 누적된 전투 데이터 초기화 없이 그대로 유지)

## 3. 유저 플로우 (플레이어가 뭘 하는가?)

1. 최종 보스 처치 후 결과 화면 등에서 '재돌입 결의'를 선택한다.
    
2. 메인 화면 상단의 타이머가 '점수판'으로 변환된다. (불가능할 경우 타이머 옆에 점수 UI가 추가로 뜬다)
    
3. 기존 전투 상태를 유지한 채 무한 모드 전용 몬스터들과 전투를 시작한다.
    
4. 몬스터 처치 시(체력이 0 이하가 될 때) 누적 처치 점수가 실시간으로 우상단 UI에 합산되어 보여진다.
    
5. 플레이어가 사망하면 즉시 게임오버가 되며, 최종 누적 처치 점수와 생존 시간 데이터가 표기된 결과 창이 뜬다.
    
6. 로컬 최고 기록과 비교하여 신기록일 경우 점수가 갱신되며, 메인 로비의 랭킹 창을 열어 1~100위 리스트와 자신의 순위(하단 고정)를 확인한다.

## 4. 연출 및 피드백 (시각/청각)

- **시각(UI/VFX):** - 진입 시 타이머가 점수판으로 자연스럽게 교체되는 연출 (미정)
    
    - 랭킹 리스트 1, 2, 3위는 좌측에 특수 메달 아이콘으로 강조 표시
        
    - 내 순위는 랭킹 리스트 최하단에 별도 패널로 고정 표출되며, 100위 밖일 경우 순위 항목에 '-' 혹은 '순위 밖' 텍스트로 표시
        
- **청각(SFX):** - 신기록 갱신 및 결과 창 출력 시 전용 사운드 재생 (미정)

## 6. 예외 상황 (~하면 어떻게 되나?)

- **비정상적인 점수 획득 (어뷰징):** 주어진 플레이 생존 시간 내에 시스템상 절대 달성할 수 없는 몬스터 처치 점수가 산출될 경우? → 점수를 무효 처리하고 랭킹에 반영하지 않음.
    
- **네트워크 단절:** 결과 창에서 신기록 달성 후 랭킹을 갱신하려는데 인터넷이 끊겨있다면? → 로컬에 최고 점수를 임시 저장해두고, 다음 메인 화면 진입 시 네트워크가 연결되어 있다면 재갱신 시도.
    
- **랭킹 데이터 로드 지연:** 랭킹 창을 열었는데 통신 지연으로 1~100위 데이터를 못 가져온다면? → 무한 로딩 빙글빙글 UI를 띄우고, 5초 초과 시 "랭킹 정보를 불러올 수 없습니다." 시스템 메시지 출력 후 빈 창 표출.
