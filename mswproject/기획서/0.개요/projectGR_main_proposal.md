## 1. 기획 의도

> **프로젝트 GR** 탄막 슈팅 + 소울 라이크 장르와 차별화된 무기 성장 시스템, 개성 있는 컨셉의 무기들, 그리고 태그 시스템으로 차별화된 전투 경험을 제공한다.

## 2. 개요

- **장르:** 액션 슈팅 어드벤처
- **플랫폼:** PC
- **시점:** 2D 탑뷰 (쿼터뷰 + 유저 캐릭터 중심 고정 카메라)

### 핵심 컨셉

- 2명의 캐릭터를 교체하며 다양한 컨셉의 무기들을 성장시키고, 몬스터와 보스들을 무찔러 연구실을 탈출한다.

### 핵심 재미

- **개성 있는 전투:** 성장 가능한 다양한 무기들을 활용하여 유저마다 개성 있는 전투 방식을 경험.
- **전략적 사고:** 시스템적으로 제공하는 일명 '쉬는 시간(유저가 시간을 정지)'으로 인해 전략적인 사고와 선택을 유도.

## 3. 중심 플로우 차트

1. **인게임 플레이 시작**
2. **준비:** 2인 파티 + 8종 무기 제공
3. **조우:** 몬스터 및 보스 조우
4. **전투:** 전투 진행
5. **성장:** 2인 태그 + 선택 무기 성장 + 패시브 획득
6. **보상:** 보상 획득 (골드, 경험치, 무기 경험치 ) + 스테이지 이동
7. **결과:**

   - 게임 기록 랭킹 등록
   - 게임 오버 시 종료 또는 재도전


## 4. 컨셉 및 세계관

### 중심 테마

- **배경:** 메이플스토리 월드
- **공간:** 알 수 없는 실험실
- **상황:** 정신을 차려보니 어두운 실험실. 서로 모르는 두 명(A와 B)이 힘을 합쳐 이곳을 빠져나가기 위해 실험실을 탈출하는 상황. 실험실을 통제하는 의문의 존재가 탈출을 막기 위해 몬스터와 보스들을 소환함.

### 캐릭터

#### 캐릭터 A (여성)

- **비주얼:** 금발
- **성격:** 좋지 않고 어두운 상황에서도 희망을 잃지 않고 활발함. 명랑하고 가볍고 밝은 느낌.

#### 캐릭터 B (남성)

- **비주얼:** 흑발
- **성격:** 비관적이고 냉소적인 성격에 쉽게 흥분하지 않음. 진중하고 무겁고 어두운 느낌.

### 비주얼 아트

- **화풍:** 메이플스토리형 도트 아트

## 5. 핵심 시스템

### 태그 시스템 (Tag System)

- **파티 구성:** 2인 1조의 파티를 한 번에 한 캐릭터씩 조종.
- **태그 스킬:** 교체 시 강력한 등장 스킬을 시전 (효과 밸런스를 고려하여 쿨타임 배정).
- **기획 의도:** 기존 장르에 태그 스킬이라는 차별화 요소를 추가하여, 빠르고 쉼 없는 게임에 '쉬어가는 구간'을 부여하고 전략적 요소를 추가함.

### 캐릭터 성장 시스템

- **성장 원리:**

  - 두 캐릭터는 **플레이어 레벨을 공유**.
  - 몬스터 처치 시 플레이어 경험치 획득.
- **레벨 업 효과:** (레퍼런스: 메이플스토리 레벨업 효과)

  - 최대 체력 증가
  - 체력 100% 회복
  - 근처 몬스터에게 광역 피해
  - 현재 논의 필요 (효과는 미정, 다만 개발 환경 구축 시 염두)

### 전투 및 무기 시스템

- **기본 전투:**

  - 탑뷰 시점에서 이동/공격하며 주변 스폰 몬스터 처치.
  - 마우스 에임 + 클릭으로 수동 공격.
- **무기 성장:**

  - 모든 무기는 적 피격 시 **"무기 경험치"** 획득.
  - 사용 빈도에 따른 성장 시스템 채택 (특정 무기에 대한 애정과 개성 표현).
- **무기 교체 (전략적 요소):**

  - **F키**를 누르면 해당 캐릭터의 보유 무기 팝업 제공.
  - **시간 정지:** 무기 팝업 트리거 시 게임 시간이 정지됨.
  - **UI:** 롤(LoL)의 핑 UI 참고 (토글 형식으로 변경 가능).
  - 설치형(포탑) 및 투사체는 무기 교체 시에도 유지됨.
- **무기 제약:**

  - 무기마다 마나를 충전해야 작동 가능.
  - 과열 방지를 위해 사용 후 일정 시간 쿨타임(사용 불가) 존재.

### 패시브 시스템

- **수급처:** 상점, 보스, 특정 몬스터.
- **효과:** 패시브 효과로 캐릭터를 추가로 강화.

### 경제 시스템

- **보상 획득:**

  - 몬스터 처치: 골드, 플레이어 경험치, 사용 무기 경험치.
  - 상자 파괴 (맵 랜덤 스폰): 골드, 체력 회복 아이템.
  - 보스/엘리트 처치: 대량의 보상.
- **상점:**

  - 맵에서 랜덤하게 등장.
  - 골드를 이용하여 회복 아이템, 탄창 아이템, 패시브 등을 구매.

### 체력 및 게임 오버

- **체력:** 캐릭터별 개별 HP바 보유.
- **리타이어:** HP가 0이 되면 해당 캐릭터 리타이어 -> 남은 캐릭터로만 강제 플레이.
- **게임 오버:** 두 캐릭터 모두 리타이어 시 게임 오버.

### 랭킹 시스템

- 최종 보스 클리어 시간 랭킹.
- 처치한 몬스터 수 랭킹.

## 6. 콘텐츠 상세

### 무기 타입

1. **단일:** 투사체 O. 몬스터 히트 시 사라짐.
2. **스마이트:** 투사체 X. 단일 즉발 데미지.
3. **광역:** 투사체 X. 면 방식 데미지.
4. **착탄 후 광역:** 투사체 O. 착탄 지점에서 광역 피해.

### 캐릭터별 무기 구성 (예시)

- **캐릭터 A (여성)**

  1. **활:** 단일, 연사 좋음, 발당 데미지 낮음.
  2. **캐논:** 광역, 연사 나쁨, 발당 데미지 높음.
  3. **포탑:** 미니건, 탄막 형성. 범위 내 자동 공격. (물리 효과/체력 없음, 일정 시간 후 파괴)
  4. **데스페라도:** 광역, 지정된 각도 범위 전체 타격.
- **캐릭터 B (남성)**

  1. **권총**: (단일, 연사 나쁨, 발당 데미지 높음)
  2. **파이어 볼:** 광역, 연사 보통, 발당 데미지 보통.
  3. **포탑:** 포탑(저격, 스마이트(지정공격)) 범위 내 자동공격, 물리 효과 없고, 체력 없고 일정 시간 후 파괴
  4. **궁니르:** 단일, 매우 높은 데미지.

### 스테이지 구성

- **총 스테이지:** 2개
- **진행 흐름:**

  1. 게임 시작 ~ 15분: 1스테이지 진행.
  2. **15분 경과:** **중간 보스** 등장 -> 처치 시 2스테이지 진입 (맵 변경).
  3. ~ 30분: 2스테이지 진행.
  4. **30분 경과:** **최종 보스** 등장.
- **클리어 후:**

  - 보스 처치 기록 랭킹 등록.
  - 또는 **무한 모드** 진입 (난이도가 기하급수적으로 상승, 몬스터 처치 수 기록 도전).
